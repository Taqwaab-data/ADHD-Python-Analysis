*ADHD Data Analysis Project:
this project analyzes ADHD- related data using Python, Pandas, and Matplotlib, and includes a full presentation explaning the insights.

*About the project:
the dataset contains informaion related to stress level, learning difficulties, task-organizing difficulties, family history, sleep patterns and more. the analysis aims to understand patterns and relationships that may affect ADHD tendencies.

*Project files:
Full Python analysis, original dataset, presentation summarizing insights

*Tools used:
Python, Pandas, tableau, powepoint, excel
